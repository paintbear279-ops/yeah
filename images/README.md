## Image Specifications (Recommended)

- **Logos**: PNG format, transparent background, ~200px width
- **Hero Images**: PNG/JPG, 1200px+ width, optimized for web
- **Icons**: PNG format, transparent background, 60-80px square
- **Asset Icons**: PNG format, 80-100px square

All images should be optimized for web to ensure fast loading times.
